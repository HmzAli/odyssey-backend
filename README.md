# odyssey-backend

## Requirements
* PostgresSQL (create a new database called 'odyssey-test')

## Installation & usage
1. clone the repo
2. `nvm use` (install nvm if you don't have it or use node version <= `v13.14.0` )
3. `npm install`
4. `npm run migrate`
5. `npm start`
6. Go to http://localhost:3000 for API info
